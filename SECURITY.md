Security Policy
Supported Versions

This project is a prototype/demo application.
Only the latest version on the main branch is supported.

Security Considerations
Do not commit API keys or sensitive credentials to the repository.
Use environment variables for any API integrations.
Mock data should be used where possible during development.
Validate all inputs in backend routes to prevent invalid or unsafe data.
Avoid exposing internal logic or system details in frontend responses.
Reporting a Vulnerability

If any security issue is found:

Report it directly to the project maintainers.
Do not publicly disclose the issue until it is reviewed.
Provide clear steps to reproduce the issue.
